Flexbox Properties

Hey, you are reading different FlexBox Properties on this page!!

flex-grow

It specifies how much the item will grow compared to the other flexible-items.

flex-shrink

It specifies how much the item will shrink compared to the other flexible-items.

flex-basis

It is used to set the length of the flex-item. Its values can be auto, inherit, length units.
